# Project PRD: Bistro im Harz - Premium Dining Experience

## 1. Project Overview
**Bistro im Harz** is a high-end, high-energy dining brand located in Osterwieck. This project aims to transform their traditional hospitality roots into a "neon-noir" digital experience. The platform serves as both a brand showcase and a functional tool for menu discovery and table reservations.

## 2. Design Vision & Brand Identity
The aesthetic is defined by the **Lux Velvet** design system, focusing on high contrast, luxury textures, and vibrant accents.

*   **Color Palette:**
    *   **Background:** Deep Velvet Black (#0A0A0A)
    *   **Primary Accents:** Emerald Green (#0F5132) & Vibrant Fuchsia Neon (#FF2A5F)
    *   **Highlights:** Champagne Gold (#D4AF37)
*   **Typography:** Google Font "Outfit" (Geometric Sans-Serif)
*   **Visual Style:** Heavy glassmorphism (backdrop blurs, subtle borders), editorial layouts, and premium scannability.

## 3. Core Features & Functional Requirements

### 3.1 Hero & Information Architecture
*   **Immersive Hero:** High-impact editorial header with the restaurant name.
*   **Operational Badges:** Floating glass panels for Location, Hours, and Contact.
*   **Primary CTAs:** "Explore Menu" (anchor scroll) and "Reserve a Table" (modal trigger).

### 3.2 Dynamic Menu Discovery
*   **Category Switching:** Tab-based navigation (Döner & Dürum, Pizza, Schnitzel, Signature Beverages).
*   **Item Mapping:** Structured data nodes for Title, Description, and Price.
*   **Interaction:** Smooth fade transitions between categories using Tailwind opacity utilities.

### 3.3 Reservation System
*   **Premium Modal:** A focused booking form with a `backdrop-blur-md` overlay.
*   **Quick Summary Widget:** A sticky sidebar/drawer for real-time booking previews.

## 4. Technical Specifications
*   **Framework:** HTML5, Tailwind CSS, Vanilla JavaScript.
*   **Responsiveness:** Fluid grid layouts (2 cols mobile, 4 cols desktop).
*   **Performance:** Optimized asset loading and CSS-only glassmorphism for performance.

## 5. Content Assets
*   **Reference Material:** Physical menu scans (Images 3, 4, 5, 6, 7) providing the source for item names, descriptions, and pricing.
*   **Logo/Iconography:** Custom Champagne Gold wordmarks and neon-themed interactive icons.

## 6. Success Metrics
*   **Aesthetic Cohesion:** Consistent application of the Lux Velvet system.
*   **Conversion:** Intuitive path to the "Reserve a Table" action.
*   **Scannability:** High readability of menu prices and descriptions against dark backgrounds.
